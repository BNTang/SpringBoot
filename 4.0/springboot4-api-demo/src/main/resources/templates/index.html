版本控制机制说明

HTTP Header方式
curl -H "API-Version: 2" http://localhost:8080/api/user
请求参数方式
curl http://localhost:8080/api/user?version=2
提示：Spring Boot 4.0 的版本控制支持多种方式，包括HTTP Header、请求参数、URL路径等，开发者可以根据需要灵活选择。
